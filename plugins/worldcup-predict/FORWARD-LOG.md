# worldcup-predict — Forward Log（開賽前鎖價驗證）

唯一乾淨的驗證方式：**開賽前**用真實運彩盤口鎖定預測，**開賽後**才結算。回測不算（運彩歷史盤口不可回溯、且用來調規則的比賽是 in-sample）。

## 紀律
- **鎖價時機**：kickoff 前記下 pick + 當下運彩賠率 + 我的機率。開賽後**不得修改**鎖定內容。
- **記 CLV**：若能取得，補記該 leg 的**收盤運彩賠率**。我的鎖定價 < 收盤價 → 正 CLV（領先市場的最佳即時證據，先於結果就知道）。
- **結算**：只評分「開賽前真正鎖定」的 claim；不事後追認沒喊過的 flag。
- **過程 vs 運氣**：判「路徑」不只看比分。晚段變異/紅牌/VAR 造成的結果算 `result hit / process partial`。
- **不因單場重新校準**：足球有不可消除的運氣底線；一場命中/落空都不當方法對錯的證據。累積 + CLV 才數。
- **in/out-of-sample**：只有開賽前鎖定 = out-of-sample。用來建規則的比賽 = in-sample，別當命中率報。

## 詳細紀錄

### #1 — 墨西哥 vs 厄瓜多 · 世界盃 R32 · 2026-07-01 10:00 (GMT+8) · Estadio Azteca

**鎖定內容（開賽前,~09:22 GMT+8,KO 前）**
- Headline：勢均力敵 / 和局機率高（墨西哥主場微幅領先）；最可能比分 **1-0 墨**（次 0-0 / 1-1）；信心 Low–Medium。
- 90 分鐘機率：**墨 42%±5 / 和 35%±5 / 厄 23%±5**。
- 關鍵判讀修正：爬蟲讀到的不讓分欄位反向，用**隊伍總分線(墨1.5 vs 厄0.5)+第1球+正確進球數**三重驗證 → 確認**墨西哥才是熱門**（1.64）。
- 運彩鎖定盤口（已轉場中,讓分/波膽/半全場/BTTS 已下架）：
  - 不讓分：墨 **1.64** / 和 **2.62** / 厄 **3.88**（去水 墨49/和31/厄21；overround ~25%）
  - 大小 1.5：大 **1.52** / 小 **1.94**
  - 墨隊總分 大小1.5：大2.20/小1.40；厄隊總分 大小0.5：大1.76/小1.66
- **下注建議：跳過。** 全盤 -EV：低比分論點被大小球 ~17% 抽水吃掉；墨 1.64 是超抽水熱門(隱含61% vs 公平~44%)、強烈 fade,但逆向的和/厄在 25% overround 下仍 -EV。娛樂逆向 lean（皆 -EV,非推薦）：和2.62 / Under1.5@1.94。

**結果（結算）**
- **墨西哥 2-0 厄瓜多**。Quinones 22'、Jimenez 31'（**兩球皆上半場**）；Hincapie 90+5' 紅牌（太晚,無影響）；開賽因雷雨延誤 1 小時。

**結算與評分**
- **下注損益：0**（照建議跳過,無下注）。**紀律過關**——運彩確實無 +EV,skip 正確；若下了娛樂 lean（和/厄/Under1.5）三個全輸。
- **預測命中**：贏家方向 ✅（墨,且是修正反向盤後正確指認的熱門）；比分 1-0 vs 2-0（同側、差一球）；**敘事 ❌**（喊「勢均力敵/和局高」,實際 31 分鐘 2-0 收工,毫不緊繃）；「贏≠掌控」「進球偏晚」皆 ❌。Under 1.5 lean 會掛（2 球）。
- **過程 vs 運氣**：**乾淨的掌控式勝利,非變異**（兩球上半場、無 VAR/點球運氣、紅牌太晚）→ 屬**真實的邊際判讀錯**,不是運氣背鍋。
- **病灶**：對已指認的共識熱門硬逆風、divergence thesis 不夠硬,還把墨壓到 **42%(低於國際去水43%、運彩去水49%)**;踩中 skill 明文警告「別只因多路徑就 headline 和局」。
- **CLV**：n/a——該場開賽前已轉場中,未能捕捉收盤前的 pre-match 運彩收盤價。往後場次務必在**接近 KO 再抓一次盤**以記 CLV。

## 下一筆模板（複製使用）

```
### #N — <主> vs <客> · <stage> · <日期 時間 GMT+8> · <venue>
**鎖定內容（開賽前,<抓取時間> GMT+8）**
- Headline / 最可能比分 / 信心 / 90' 機率(墨X 和Y 厄Z ±band)
- 運彩鎖定盤口：不讓分 / 讓分 / 大小 / 波膽 / 隊總分 …（標來源與時間）
- 下注建議：<BET/lean/跳過> · pick @ 運彩賠率 · 注碼 · EV
- (KO 前再抓一次) 收盤運彩賠率 → 記 CLV
**結果（結算）**：比分 / 進球時間 / 紅牌 / 特殊事件
**結算與評分**：下注損益 / 預測命中 / 過程vs運氣 / CLV / 病灶或亮點
```
